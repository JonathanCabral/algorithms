**Busca Binária**
-

***Exercicios/Anotações***

**1 -** Suponha que você tenha uma lista com 128 nomes e esteja fazendo uma 
pesquisa binária. Qual seria o número máximo de etapas que voçê levaria para 
encontrar o nome desejado?

**R:** 7 etapas. Log2 128 = 7

**2 -** Suponha que voçê duplique a lista de tamanho. Qual seria o número máximo de etapas agora?

**R:** 8 Etapas. Log2 256 = 8
